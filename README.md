# AIPowered_ResumeAnalyzer
An intelligent web application built using Python, Flask, Machine Learning, and NLP that analyzes resumes and provides actionable insights. This tool automatically extracts key skills, identifies missing keywords, and gives an ATS-friendly score to help job seekers optimize their resume for modern hiring systems.

The system uses NLP-based text processing, ML models, and integrated AI APIs to understand resume content, highlight strengths, and recommend improvements. The clean web interface is fully built with Flask, making it easy to upload resumes, process them, and display detailed analysis results instantly.
